## Talkora Forum

Forum discussion application for students to ask questions and get answers from other students and teachers.

### Features

- User authentication (login, register, logout)
- Thread creation and management
- Comment system
- Upvote/downvote system
- User profile
- Search functionality
- Responsive design

### Tech Stack

- React
- Vite
- React Router
- Redux Toolkit
- Cypress
- Vitest

### Installation

```bash
# Clone the repository
git clone https://github.com/febss20/Talkora-Forum.git

# Navigate to the project directory
cd forum-app

# Install dependencies
npm install

# Run the development server
npm run dev
```

### Running Tests

```bash
# Run unit and integration tests
npm run test

# Run end-to-end tests
npm run e2e
```
